# Functions

Functions must be defined as camelCase

`myFunction`,
`myFunction2`,
`printHelloWorld`,

## Define a function

```C
int :: main(int argc, string[] argv) {
    int variable = 40
    print("Hello World! $(variable)\n")
    
    return 0
}
```

## Call a function

```C
int :: add(int a, int b) {
    return a + b
}

add(1, 2) // Note, functions must be called inside a scope
```
